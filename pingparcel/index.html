FAQ

Is this an AI chatbot? +

PingParcel isn’t a generic chatbot. It’s a focused post-purchase layer for shipping and returns updates. We start with explainable, policy-aware replies and only add generation where it helps.

What platforms do you support? +

We’re starting with Shopify. Others may follow if we can keep it simple and reliable.

When can I try it? +

We’re inviting a handful of stores to the private preview. Request access above and we’ll be in touch.

Do I need to migrate my help desk? +

No migration required. PingParcel is designed to slot in alongside your current setup.
